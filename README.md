# Professional-DOCX-Merger
Merge multiple DOCX files in numeric order, insert page breaks between them, preserve images and styles.

- Preserving **images, tables, and formatting**
- Inserting **page breaks** between merged documents
- Respecting **numeric order** of file names
- Displaying a **progress bar** in the terminal
